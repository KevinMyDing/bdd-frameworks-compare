BDD 框架对比: Cucumber.js vs Robot Framework vs Gauge.js
===

Requirements: Java 8, Node.js 8

Cucumber.js
--

```
yarn install
```

```
npm test
```


Robot
---

```
pip install -r requirements.txt
```

```
robot robot_tests
```

Gauge
---

```
brew install gauge
```

```
$ gauge init -t
```
